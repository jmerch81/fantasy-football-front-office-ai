# Fantasy Football Front Office AI

Fantasy Football Front Office AI is an AI-powered fantasy football decision intelligence platform designed to help users manage fantasy teams through data-driven recommendations, predictive analytics, machine learning, and LLM-powered coaching.

## Project Goals

This project is being built as a production-style portfolio project to demonstrate skills in:

- Python
- SQL
- Data Engineering
- Machine Learning
- Recommendation Systems
- Predictive Analytics
- APIs
- Streamlit Dashboards
- Feature Engineering
- LLM Integration
- RAG
- Software Engineering Best Practices

## Planned Features

- AI Draft Assistant
- Weekly Lineup Optimizer
- Waiver Wire Advisor
- Trade Analyzer
- Championship Probability Simulator
- Player Similarity Engine
- Injury Impact Analysis
- Rest-of-Season Rankings
- AI Fantasy Coach
- Team Power Rankings
- Strength of Schedule Analysis
- Weekly Projections Dashboard
